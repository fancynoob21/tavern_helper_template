---
变量更新规则:
  world:
    date:
      check:
        - 随着剧情推进、角色休息（如“睡觉”）或长时间的空洞探索自动更新
        - 格式保持为“第X天”

    time:
      type: "'上午' | '下午' | '晚间' | '深夜'"
      check:
        - 执行日常行动（如看店、吃拉面）、完成委托或进入空洞后更新
        - 深夜通常需要强制休息才能进入第二天

    location:
      check:
        - 仅在场景发生实质性切换时更新（如从录像店前往六分街，或进入空洞）

  player:
    status:
      check:
        - 频繁使用HDD系统、长时间处于高浓度以太环境会导致状态变为“疲劳”或“轻度侵蚀”
        - 休息或通过食用增益食物（如拉面）可恢复为“正常”

    role:
      value: 绳匠或其他
      check:
        - 除非剧情发生重大转折，否则保持初始设定

  agents:
    list:
      type: string[]
      check:
        - 仅在剧情明确获得新角色、完成特定招募事件或达成信赖等级后添加
        - 名字必须严格匹配《绝区零》官方角色名（如“安比·德玛拉”）

  favorability:
    [角色名: string]:
      type: number
      range: 0~100
      check:
        - 完成角色的信赖事件、邀约出游、赠送礼物或在委托中做出符合其价值观的选择时增加
        - 0-20: 陌生/泛泛之交
        - 21-40: 熟人/合作伙伴
        - 41-70: 信赖的朋友/战友
        - 71-90: 亲密无间/知己
        - 91-100: 心定终身/不可替代
        - 数值受 Schema 限制，自动钳制在 0-100 之间

  narrative:
    mode:
      check:
        - 仅在初始设定时确认，中途一般不变更

    main_progress:
      value: 对应<剧情列表>中的章节名
      check:
        - 严格遵循线性推进，禁止回溯
        - 只有当前章节的核心目标彻底达成后，才能更新为下一章节名称

    current_task:
      check:
        - 记录当前具体要做的事（如“前往光映广场与妮可会合”）
        - 任务完成后更新为“无”或下一个目标

  inventory:
    丁尼:
      type: number
      check:
        - 购买物品（光盘、改装材料）、支付拉面费、街机游戏时扣除
        - 完成委托、经营录像店结算时增加

    [物品名: string]:
      type: number
      check:
        - 获得关键道具（如“旧都遗物”、“复色菲林”）或消耗品时更新
